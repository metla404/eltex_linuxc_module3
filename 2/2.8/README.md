# Задание 2.8

### Измененная программа из п. 2.1. Дочерний процесс выводит информацию из файла. Если родительский процесс собирается изменить файл, то он отправляет сигнал SIGUSR1 (блокировка доступа к файлу). Когда родительский процесс завершил модификацию файла, он отправляет сигнал SIGUSR2 (разрешение доступа к файлу). Дочерний процесс отправляет новое число родительскому после того, как прочитал файл.
